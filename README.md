### about

I work around systems engineering and cryptography.

My interest is in building security-critical software that is clear, measurable, and hard to misuse. I care about how cryptographic ideas become real systems: protocols, implementation safety, verification, low-level constraints, and real-world behavior.

Cryptography research is part of my direction, especially when it connects theory with implementation.
